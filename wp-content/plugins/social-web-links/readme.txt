Display your social profile in widget, post content anywhere.

== Description ==

This plugin will display elegant icons of your social profiles on a sidebar widget. Plugin also provides options to upload your own icons if you do not like the default ones. 4 Size options and 5 display styles available.

== Frequently Asked Questions ==

= How do I add my own icons ? =

In the admin page, custom social sites tab will let you upload/select your own icons. In order to display it, you will have to select the icon size setting to be the same as your uploaded icon size. For e.g. if you uploaded an icon of 16x16 pixels and you are select the icon size setting dropdown as 24x24 pixels, then your icon will not be displayed

= How do I display my social profile icons in areas like Header or Footers ?

To display social profile icons in areas other than post/page content and widgets. You can enter below shown code at the location where you want the icons to be displayed. The code is:-

<?php swl_display(); ?>

= Does my custom icons override default icons ? =

Yes, if you are displaying the same size as your uploaded custom icon

= Will resetting the settings in admin page, delete all my data ? =

Yes, you will lose all your URLs and custom icons you entered
